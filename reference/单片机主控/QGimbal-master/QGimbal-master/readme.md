# QGimbal高性能二维云台

## 简介

QGimbal二维云台稳定器，使用大疆C板(stm32f407)作为主控芯片控制两个电机，实现双轴电流（力矩）控制、速度控制和角度控制，并读取陀螺仪数据进行姿态解算，实现类似相机稳定器的增稳效果。

QGimbal官网：https://qdrive.com.cn/products/qgimbal/

QGimbal效果展示：https://www.bilibili.com/video/BV1MM22B4Ebq/

## 特性

- 高性能：基于STM32F407系列MCU，提供强大的处理能力。
- 高精度：使用无刷电机作为底层驱动器，具有精度高，响应快的特点。
- 宽电压：支持8-24V的电压输入，2-6S电池适用。
- 集成IMU: 内置BMI088六轴IMU，AHRS姿态解算，提供高精度的姿态信息。
- ms级响应：经测试，可在100ms内使云台转动半圈。
- 丰富接口：支持UART通信、USB控制、支持shell交互。
- 多种控制模式：支持位置控制、速度控制和电流控制。支持角度步进模式、低速高转矩模式。
- 参数可调：提供丰富的参数设置选项，包括三环PID参数、零点设置、波特率设置等。
- 易于使用：提供详细的文档和示例代码，帮助用户快速上手和二次开发。
