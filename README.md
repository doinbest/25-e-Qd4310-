# 2025 年电赛 E 题 QD4310 无刷电机项目

## 项目简介

本项目用于 2025 年全国大学生电子设计竞赛 E 题相关开发。

系统由 STM32 主控、QD4310 无刷电机、MaixCAM2 视觉模块及相关传感器组成。
STM32 负责模式管理、控制算法、安全保护和电机执行；
MaixCAM2 负责视觉识别，并通过串口向 STM32 发送识别结果。
